Name Path HTTP Verb Purpose
Index /blog GET List of all blog

Create /blog POST Create a new Dog, then redirect somewhere

Edit /blog/:id/edit GET Show edit form for one blog information
Update /blog/:id PUT Update a particular blog , then redirect to somewhere
Destroy /blog/:id DELETE Delete a particular blog, then redirect to some where
